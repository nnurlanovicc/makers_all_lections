1. pwd - (print working directory) -> возвращает абсолютный путь до нашей директории
2. ls - (list) -> выводит содержимое директории (текущей или указанной)
3. cd - (change directory) -> перемещает в домашнюю директорию
  cd ../ cd - -> возвращает на директорию выше
  сd / -> перемещает в корневую директорию (/) 
  cd <dir_name> -> перемещает в указанную директорию (можно 
  использовать абсолютный путь )
4. mkdir <dir_name> -> создает указанную директорию
5. clear -> очищает терминал (ctrl + l -> смещает)
6. touch <file_name> -> создает указанный файл
7. rmdir <dir_name> -> удаляет указанную пустую директорию
8. rm <file_name> -> удаляет  файл 
9. rm -rf <dir_name> (remove recursiv force) -> удаляет указанную директорию
 со всем содержимым
10. nano <file_name> -> открывает/создает файл в текстовом редакторе
11. cat <file_name> (concatenate) -> выводит содержимое файла в терминал 
12. mv <file_name1> <file_name2> -> перемещает/переименовывает файл
13. cp <file_name1> <dir_name> -> копирует file_name1 в директорию dir_name
14. find -> ищет указанный файл или директорию 
  (универсальный вариант find /home -name <name>)
15. history -> выводит историю команд
16. sudo -> передоставляет права админа
17. tree -> выводит дерево содержимого текущей директории
18. echo text > file_name -> записывает text в уазанный файл
19. cat file_name > file_name2 -> копирует содержимое file_name в file_name2
20. du <dir_name> (disk usage) -> показывает кол-во занимаемой памяти
